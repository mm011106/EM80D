# EM-80D/R HDLコード

## 概要
- DR20000110-07　に対応するHDLコード (VHDL)
- 動作開始時にトリガ前のタイミングでch0をドライブする、というバグがあり、それに対処


## 仕様
- 5ch, 24波バースト（マーカコイルドライバの標準仕様）
- SYNC = CH0,　第0波で、1波分アサート
- ReTHMのコントロール可能

## 対応デバイス
- MAX V 5M80ZT100
